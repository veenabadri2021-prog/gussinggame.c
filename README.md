## Projects

### 1. Guessing Game (gussinggame.c)

### 2. Simple Calculator (simplecalculator.c)

### 3. NagarVaani Project

An AI-powered civic grievance resolution system for India. Citizens can file complaints via WhatsApp voice notes in Hindi or regional languages, which are automatically verified, prioritized, and routed to the correct government department. Built for ANANTYA 2026 at NIT Raipur.

### 4. AI Driven Cross Platform Recommendation System


A research paper published in JETIR (March 2026) proposing an AI-powered recommendation system that analyzes user behavior and trends across multiple platforms. Uses NLP, Large Language Models, and collaborative filtering to deliver personalized recommendations — demonstrated through Netflix and Amazon case studies.
